وكالة الدواحي للسفر والسياحة — التشغيل من داخل الهاتف (Termux)
================================================================

الآن المشروع بيه قاعدة بيانات حقيقية (SQLite) بتتخزن كملف "dawahi.db"
داخل نفس مجلد المشروع على هاتفك — مش نسخة عرض وهمية زي قبل.

خطوات التشغيل:
---------------

1) افتح Termux وثبّت بايثون (لو ما مثبت):
   pkg update && pkg install python -y

2) ثبّت المكتبات المطلوبة:
   pip install flask werkzeug

3) انسخ مجلد المشروع كامل إلى الهاتف (فك الضغط)، ثم ادخل عليه:
   cd مسار/المجلد

4) شغّل السيرفر:
   python app.py

5) هتظهر لك رسالة فيها رابط، افتحه من متصفح الهاتف نفسه:
   http://127.0.0.1:5000

ملاحظات مهمة:
--------------
- أول مرة تشغّل السيرفر، هيتم إنشاء ملف "dawahi.db" تلقائياً — دي قاعدة
  بياناتك الحقيقية، وكل حساب أو حجز بيتسجل فيها فعلياً.
- صور جوازات السفر بتتخزن داخل مجلد uploads/passports.
- عشان تفتح الموقع من جهاز تاني على نفس الشبكة (واي فاي)، استخدم رقم
  الآيبي بتاع الهاتف بدل 127.0.0.1 (مثلاً http://192.168.1.5:5000).
- لو حابب تشوف محتوى قاعدة البيانات مباشرة، ثبّت أداة "DB Browser for
  SQLite" على الكمبيوتر وافتح ملف dawahi.db، أو استخدم من Termux:
      pip install sqlite-web
      sqlite_web dawahi.db
- عشان توقف السيرفر: اضغط Ctrl + C داخل Termux.
- لو عايز تبدأ بقاعدة بيانات فاضية من جديد، احذف ملف dawahi.db وشغّل
  app.py تاني، وهيتعمل من الصفر تلقائياً.

هيكل القاعدة:
--------------
جدول users:  الحسابات (اسم المستخدم، البريد، الهاتف، بيانات الجواز،
              كلمة المرور مشفّرة بـ werkzeug — مش نص عادي).
جدول bookings: الحجوزات (طيران/فنادق) مرتبطة بكل مستخدم برقم حجز فريد.
